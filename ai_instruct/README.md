# ai_instruct

This folder is for documentation, tasks, features, and a central file for agent collaboration, delegation, and updates. All agents and developers should use this as the main coordination hub for Eliza development.

## Structure
- `tasks.md`: Ongoing and completed tasks
- `features.md`: Feature requests and specs
- `collab.md`: Agent and developer updates, questions, and delegations
- `README.md`: This file
